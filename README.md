# Projeto Farmácia
Um sistema de integração entre farmácia e hospital que permite aos médicos enviar receitas diretamente para a farmácia, agilizando o processo de retirada de medicamentos pelos pacientes.

O sistema será utilizado diretamente por médicos no hospital para enviar receitas eletrônicas para a farmácia credenciada. Os pacientes poderão retirar seus medicamentos na farmácia mediante apresentação 
da receita eletrônica e de um documento de identificação.
